Ping pong

By Tina Liu

Overview

In this project, I recreated a traditional pong game. I considered UX/UI designs and wanted to improve an initial project. With processing, I was able to develop a visually appealing game while also improving my game development skills.

Tools

Objective

In my spare time, I wanted to improve an exisitng pong game that I created for COSC123. I felt that I had improved a lot after a year of learning Java, so I wanted to see if I could implement any UX/UI designs to improve the experience.

Development

Before doing any coding at all, I wanted to test out my existing project to see what required improvement.

*The requirements needed for this project was a score counter, a paddle, and a ball that reflects if it's within the paddle's dimensions.

The game's functionalities were very limited. First off, the ball didn't bounce correctly from the paddle. I wanted to revise this issue in the newest game. Secondly, I wanted to change the game into a 2 person game to increase the difficulty. Thirdly, I wanted to include a reset button so I wouldn't need to rerun the code if I wanted to restart. Lastly, I changed the paddle to move from the mouse to the following keyboard keys: 'UP arrow', 'DOWN arrow', 'A', and 'Z'.

Conclusion

After adding all the new elements to the game, I strengthened my understanding of game development and user experience. Overall, it was a fun project to work on and it was exciting to see the progress I made as a student.
